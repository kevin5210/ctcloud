公司介绍
程天科技是一个致力于外骨骼机器人研发的科技企业，研发团队包含科研、机械、电子、软件四个主要研发部门，具有多年的特种机器人、水下机器人等高精尖人才与技术，涵盖自动化、互联网、运动医学等多个专业。与哈工大、哈工程（哈军工）、浙江大学、加拿大卡尔顿大学等多所高校密切科研合作关系，并与哈尔滨医大、首都医科大学、黑龙江中医药大学、浙二医院、等著名医院一起合作进行外骨骼配套康复方案的研究。为行动不便和需要助力的人群提供更好的外骨骼产品。
